#### 🔸   연결이 되지 않을 경우

> 1. Git Clone으로 프로젝트 파일을 받아옵니다. <br/>
> `git clone "https://github.com/wafflestudio20-5/team10-web.git"`
> 2. toyproject 디렉토리로 이동합니다. <br/>
> `cd team10-web/toyproject`
> 3. npm이나 yarn을 이용해 실행해 줍니다. <br/>
> `npm run start` or `yarn start`
